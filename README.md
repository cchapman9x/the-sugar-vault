# The Sugar Vault
An arduino-controlled candy dispenser using a stepper motor and 3D-printed mechanical design.

<hr>

### Project Mission Summary

I have several reasons for creating the Sugar Vault. The primary reason is to enhance my engineering skills while learning about microcontrollers and how to control motors. I wanted an introductory project that I could handle and would be fun, so I came up with this. The second reason is to create something that can be placed in the office where I work so that my coworkers and I can enjoy treats while doing our job. This isn’t only an engineering project, but a more fun and personal one.

The Sugar Vault has one primary function: to dispense 3 different candies depending on the user's input. The user will request candy from the Sugar Vault by pressing 1 of 3 buttons. Each button will control a mechanism that dispenses a type of candy from 1 of 3 tubes, each filled with a different candy. The logic of the Sugar Vault will be controlled by an Arduino, which is powered either by batteries or by being plugged into an outlet. The mechanism that allows the candy to be dispensed will be an iris mechanism powered by a stepper motor. The idea is to use the Arduino to control the speed and rotation of the motor in order to open the iris mechanism and close it once the candy has been dispensed.
